# Investment Performance Dashboard

This project is an Excel-based MIS dashboard analyzing SIP and Lumpsum investment data.

## Key Features:
- Region-wise investment analysis
- Monthly trend analysis
- Interactive slicers (Region & Date)
- KPI metrics (Total Investment, Avg SIP, etc.)

## Tools Used:
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers

## Insights:
- North region has highest investment contribution
- Lumpsum investments show higher volatility compared to SIP
